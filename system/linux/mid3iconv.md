# mid3iconv

Converting ID3 tag's encoding for mp3 files.

### Installation

```shell
$ sudo dnf install python-mutagen
```

### Usage
```shell
$ mid3iconv -e [encoding] input.mp3

# dry-run and print tag

$ mid3iconv -p -d -e [encoding] input.mp3

```
